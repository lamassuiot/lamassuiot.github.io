LAMASSU IOT

A Public Key Infrastructure (PKI) for the Internet of Things (IoT) designed to run on both commodity hardware and cloud infrastructure.

Why Lamassu

What Lamassu can do

In House PKI

Flexible In-house PKI for configuring, expanding and managing certificates.

Open Source

Being an Open Source software, Lamassu IoT allows organizations to freely evolve it to cope with their specific needs.

API

PKI services are exposed via effective APIs for automation.

IoT

Lamassu IoT deals with the heterogeneity and scale of IoT deployments.

Cloud

Lamassu IoT is designed to run on both commodity hardware and cloud infrastructure.

Scalability

Lamassu IoT can be easily scaled by adding more nodes.

Fault tolerance

Hardware failure is the norm rather than the exception. Lamassu IoT offers detection of faults and quick and automatic recovery.

Exchange data
securely over
the network

Lamassu PKI IoT enables connected devices and applications to securely exchange data over networks.
Linear scalability and fault-tolerance on commodity hardware or cloud infrastructure make it the perfect solution for securing large and mission critical deployments.
